# 🐧 Linux File Management System
A hands-on Linux Administration project built on **Red Hat Enterprise Linux (RHEL)** to demonstrate real-world file and directory management, permissions, ownership, backup, and reporting.

# 📖 Project Overview

This project simulates a real-world file management environment for an organization with multiple departments.
The project demonstrates Linux system administration tasks including:

- Directory Management
- File Operations
- File Permissions
- Ownership Management
- Soft & Hard Links
- Backup & Restore
- Compression
- Disk Usage Analysis
- Report Generation

---
# 🎯 Project Objectives

- Practice Linux file and directory management
- Learn Linux administration concepts
- Implement backup and recovery
- Work with Linux permissions
- Organize files in an enterprise-like structure

# 💻 Linux Commands Used

| Command | Description |
|----------|-------------|
| mkdir | Create directories |
| touch | Create files |
| cp | Copy files |
| mv | Move or rename files |
| rm | Delete files |
| chmod | Change permissions |
| chown | Change ownership |
| find | Search files |
| grep | Search file content |
| tar | Backup |
| gzip | Compression |
| du | Disk Usage |
| df | Filesystem Usage |
| tree | Directory Structure |
