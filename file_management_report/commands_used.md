# Commands Used

| Command | Purpose |
|---------|----------|
| mkdir | Create directories |
| touch | Create files |
| ls | List files |
| cp | Copy files |
| mv | Move or rename files |
| rm | Delete files |
| chmod | Change permissions |
| chown | Change ownership |
| find | Search files |
| grep | Search text |
| tar | Create backups |
| gzip | Compress files |
| du | Disk usage |
| df | Filesystem usage |
